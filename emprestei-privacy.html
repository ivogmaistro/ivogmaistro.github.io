EMPRESTEI

Política de Privacidade

Última atualização: 06 de novembro de 2025

1. INTRODUÇÃO

A LoMa Softwares ("nós", "nosso" ou "nos") opera o aplicativo móvel Emprestei (doravante referido como "Serviço"). Esta página informa sobre nossas políticas em relação à coleta, uso e divulgação de dados pessoais quando você usa nosso Serviço e as escolhas que você tem associadas a esses dados.

Resumo: O Emprestei é projetado com a privacidade em mente. Todos os seus dados de empréstimos são armazenados localmente no seu dispositivo e você tem controle total sobre eles.

2. COLETA E USO DE INFORMAÇÕES

2.1 Dados Armazenados Localmente

O aplicativo Emprestei armazena as seguintes informações localmente no seu dispositivo:

  • Informações sobre itens emprestados (título, descrição, pessoa, datas)
  • Configurações de lembretes e notificações
  • Preferências do aplicativo (tema, configurações de interface)

Importante: Esses dados são armazenados exclusivamente no seu dispositivo e não são transmitidos para nossos servidores ou terceiros.

2.2 Dados Analíticos

Para melhorar a experiência do usuário e a qualidade do aplicativo, coletamos dados analíticos anônimos através do Google Firebase Analytics, incluindo:

  • Informações sobre como você usa o aplicativo (recursos mais utilizados)
  • Informações do dispositivo (modelo, versão do Android, idioma)
  • Dados de crashes e erros para correção de bugs
  • Dados de desempenho do aplicativo

Esses dados são anônimos e não podem ser usados para identificá-lo pessoalmente.

2.3 Permissões do Aplicativo

O aplicativo solicita as seguintes permissões:

  • Notificações (POST_NOTIFICATIONS): Para enviar lembretes sobre itens emprestados
  • Wake Lock (WAKE_LOCK): Para garantir que as notificações sejam entregues no horário programado

3. USO DOS DADOS

Utilizamos os dados coletados para:

  • Fornecer e manter nosso Serviço
  • Notificá-lo sobre mudanças em nosso Serviço
  • Permitir que você participe de recursos interativos quando escolher fazê-lo
  • Fornecer suporte ao cliente
  • Coletar análises ou informações valiosas para melhorar o Serviço
  • Monitorar o uso do Serviço
  • Detectar, prevenir e resolver problemas técnicos

4. COMPARTILHAMENTO DE DADOS

Nós não vendemos, trocamos ou alugamos seus dados pessoais para terceiros. O aplicativo Emprestei foi projetado para manter seus dados privados e seguros.

Dados analíticos anônimos podem ser compartilhados com:

  • Google Firebase: Para análise de uso e relatórios de crash

5. SEGURANÇA DOS DADOS

A segurança dos seus dados é importante para nós. Como todos os dados de empréstimos são armazenados localmente no seu dispositivo, você tem controle total sobre eles. Recomendamos:

  • Manter seu dispositivo atualizado com as últimas atualizações de segurança
  • Usar bloqueio de tela para proteger o acesso físico ao dispositivo
  • Fazer backup regularmente dos dados do seu dispositivo

6. BACKUP E EXCLUSÃO DE DADOS

Você pode excluir todos os dados do aplicativo a qualquer momento:

  • Desinstalando o aplicativo do seu dispositivo
  • Limpando os dados do aplicativo nas configurações do Android

Nota: Ao desinstalar ou limpar os dados, todos os registros de empréstimos serão permanentemente excluídos e não poderão ser recuperados.

7. PRIVACIDADE DE CRIANÇAS

Nosso Serviço não se destina a pessoas com menos de 13 anos ("Crianças"). Não coletamos intencionalmente informações pessoalmente identificáveis de pessoas com menos de 13 anos.

8. LINKS PARA OUTROS SITES

Nosso Serviço pode conter links para outros sites que não são operados por nós. Aconselhamos fortemente que você reveja a Política de Privacidade de cada site que visitar.

9. MUDANÇAS NESTA POLÍTICA DE PRIVACIDADE

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página e atualizando a data de "última atualização".

Recomendamos que você revise esta Política de Privacidade periodicamente para quaisquer alterações.

10. SEUS DIREITOS (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem os seguintes direitos:

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Informação sobre compartilhamento de dados
  • Revogação do consentimento
